# kancolle_fleet_capture

「kancolle_fleet_capture」は、ブラウザゲーム「艦隊これくしょん」の艦娘詳細を
１〜６隻までキャプチャ＆連結し、png画像として保存するFirefox用のアドオンです。

##使用方法
１. 【編成】からキャプチャしたい艦娘の【詳細】を表示させる
２. ゲーム画面の左右の白い余白を一度クリックしてから【S】キーでキャプチャ
３. ゲーム画面の左右の白い余白を一度クリックしてから【X】キーで画像を保存

６隻以上キャプチャすると、７隻目で勝手に保存されます。
【S】【X】キーを押下する時は、一度ゲーム画面左右の白い余白をクリックしてください（※上下の余白ではダメ）

##キャプチャがずれる場合
・ブラウザのズーム機能が働いていませんか？（Ctrl+0 で等倍に戻せます）
・Retina等高解像度の場合は、拡張機能オプションから別途調整してください

##仕様
・ローカルストレージを一時的に10MBほど使います（画像生成時にクリア）
・サーバ側との通信は一切しません。HTMLの変更も行いません。
